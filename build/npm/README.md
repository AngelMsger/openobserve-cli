# @angelmsger/openobserve-cli

npm distribution wrapper for **openobserve-cli** — an agent-facing CLI for
[OpenObserve](https://openobserve.ai) (O2). Installing this package downloads the
prebuilt Go binary for your platform from the matching GitHub Release.

```bash
npm install -g @angelmsger/openobserve-cli
openobserve-cli --help
```

If installed with `--ignore-scripts`, the binary is fetched lazily on first run.

See the [project README](https://github.com/AngelMsger/openobserve-cli) for usage.
